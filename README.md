# ServiceBooking Web Application

## Description
A full-stack web application where users can book services online and administrators can manage services and bookings.

## Features
- User registration and login
- Service booking system
- Admin dashboard
- Booking management
- Responsive design

## Technologies Used
- Frontend: React / HTML / CSS / Bootstrap
- Backend: Node.js / Express
- Database: MongoDB

## How to Run
1. Clone this repository
2. Install dependencies
3. Run the frontend and backend servers
